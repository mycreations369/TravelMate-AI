# TravelMate AI

## Overview
TravelMate AI is a smart travel assistant that helps users plan trips.

- Day-wise Itinerary
- Budget Breakdown
- Transport Recommendation
- Hotel Recommendation
- Food Suggestions
- Weather Advice
- Packing Checklist
- Safety Tips
- Download Travel Plan
